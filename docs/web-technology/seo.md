# SEO

SEO（搜索引擎优化）是一种用于提升网页在搜索引擎中的收录数量以及排序位置而做的优化行为。

所需的一切， 排名更高 ⭐ 并获得更多流量 ⛰️

## SEO 怎么做

SEO 是一个持续进行的过程，大致分为四个方面：

关键词研究、**页面 SEO**、**链接建设**、技术性 SEO
